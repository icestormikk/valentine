Веб-проект, разработанный в качестве милого подарка на 14 февраля для моей девушки. Представляет из себя нечто наподобие визуальной новеллы :)

## Языки и инструменты
- React (+ Typescript)
- CSS
- HTML

## Установка и запуск
1. Склонируйте репозиторий:
```bash
git clone https://github.com/icestormikk/valentine.git
```
2. Перейдите в каталог проекта:
```bash
cd valentine
```
3. Установите зависимости:
```bash
npm install
```
4. Запустите локальный сервер:
```bash
npm run dev
```
5. Открой проект в браузере по адресу, который появится в консоли
